# Скрінер та аналітика

## Screener (Скрінер)

Скрінер — це ваш інструмент для пошуку торгових можливостей серед 20 монет на 3 біржах.

### Як користуватись

1. Перейдіть в **Screener** в бічній панелі
2. Бачите таблицю з 7 колонками:

| Колонка | Що означає |
|---|---|
| **Pair** | Торгова пара (наприклад BTC/USDT) |
| **Exchange** | Біржа (Binance, Bybit, OKX) |
| **Price** | Поточна ціна |
| **24H%** | Зміна за 24 години (зелена = зростання, червона = падіння) |
| **Volume** | Обсяг торгів в USD |
| **RSI** | Індекс відносної сили (0-100) з кольоровою полоскою |
| **Trend** | Бичачий / Ведмежий / Нейтральний |

### Фільтри

- **Exchange**: All / Binance / Bybit — фільтр по біржі
- **Trend**: All / Bullish / Bearish — фільтр по тренду

### Готові скани

- **All** — показати все
- **Oversold (RSI<30)** — монети з RSI нижче 30 (потенційно дешеві)
- **Overbought (RSI>70)** — монети з RSI вище 70 (потенційно дорогі)
- **Volume Surge** — монети з найбільшим обсягом (top 25%)

### Сортування

Натисніть на заголовок будь-якої колонки щоб відсортувати:
- Перший клік — за спаданням
- Другий клік — за зростанням
- Активна колонка виділена золотим

### Пошук

Введіть назву монети в поле пошуку (наприклад "ETH") — фільтрує таблицю.

### Перехід на графік

Натисніть на будь-який рядок — відкриється графік цієї монети.

## Market Heatmap (Теплова карта)

Візуальний огляд всього ринку одним поглядом.

1. Перейдіть в **Heatmap**
2. Кожна монета — прямокутний блок
3. **Розмір блоку** пропорційний обсягу торгів
4. **Колір**: зелений = зростає, червоний = падає, інтенсивність = величина зміни
5. Натисніть на блок → відкриється графік

Перемикач "Color by":
- **Price Change** — кольори за зміною ціни
- **RSI** — кольори за значенням RSI

---

## Correlation Matrix (Кореляційна матриця)

Показує як пов'язані рухи різних монет.

1. Перейдіть в **Correlation**
2. Таблиця 20×20 з кольорами:
   - **Червоний (+1.0)** = монети рухаються однаково
   - **Сірий (0)** = немає зв'язку
   - **Синій (-1.0)** = монети рухаються протилежно
3. Діагональ завжди 1.00 (золота) — монета корелює сама з собою

**Як використовувати:**
- Для диверсифікації шукайте монети з низькою кореляцією (< 0.3)
- Якщо BTC-ETH = 0.8, вони рухаються разом — тримати обидва не дає диверсифікації

## Confluence Map (Карта збігів сигналів)

Показує цінові рівні де збігаються кілька технічних сигналів.

1. Перейдіть в **Confluence**
2. Виберіть монету
3. Бачите список цінових зон з кількістю джерел:
   - **Weak** (1 джерело) — ненадійно
   - **Moderate** (2) — варто спостерігати
   - **Strong** (3) — висока ймовірність
   - **Extreme** (4+) — найсильніший рівень

Джерела: EMA, SMA, Support/Resistance, Bollinger Bands, RSI зони, сигнали

## Pattern Scanner (Сканер патернів)

Автоматично знаходить графічні патерни:

1. Перейдіть в **Pattern Scanner**
2. Виберіть монету та таймфрейм
3. Система аналізує останні 50 свічок та знаходить:
   - **Double Top/Bottom** — фігури розвороту
   - **Higher Highs / Lower Lows** — структура тренду
   - **Doji** — невизначеність (тіло ≈ 0)
   - **Hammer** — потенційний розворот вгору
   - **Engulfing** — поглинання (сильний розворот)

Кожен патерн має:
- Тип: Bullish (зелений) / Bearish (червоний) / Neutral
- Confidence: відсоток впевненості
- Опис що означає патерн

## Advanced Patterns (Розширені патерни)

Об'єднана сторінка з 3 вкладками: Elliott Wave, Harmonic, Wyckoff.

1. Перейдіть в **Advanced Patterns** (або `/advanced-patterns`)

### Elliott Wave (Хвилі Елліотта)

Автоматичний підрахунок хвиль:

1. На вкладці **Elliott Wave** виберіть монету
2. Система шукає:
   - **Impulse** (5 хвиль): 1-2-3-4-5 з правилами (Wave 3 ≠ найкоротша)
   - **Correction** (ABC): A-B-C корекційний рух
3. Fibonacci targets: прогнозовані цілі для Wave 3 та Wave 5

### Harmonic Patterns (Гармонійні патерни)

Геометричні патерни з Fibonacci співвідношеннями:

1. Переключіть на вкладку **Harmonic**
2. Система шукає XABCD точки:
   - **Gartley**: B = 0.618 XA, D = 0.786 XA
   - **Butterfly**: B = 0.786 XA, D = 1.27 XA
   - **Bat**: B = 0.382-0.5 XA, D = 0.886 XA
   - **Crab**: B = 0.382-0.618 XA, D = 1.618 XA
3. PRZ (Potential Reversal Zone) — зона де очікується розворот

### Wyckoff Phase (Фази Вайкоффа)

Визначає фазу ринкового циклу:

1. Переключіть на вкладку **Wyckoff**
2. Виберіть монету
3. Фази:
   - **Accumulation** (зелений) — розумні гроші купують, ціна в range → готуватись до купівлі
   - **Markup** (зелений) — зростаючий тренд → тримати/купувати
   - **Distribution** (червоний) — розумні гроші продають → готуватись до продажу
   - **Markdown** (червоний) — падаючий тренд → продавати/чекати

## Seasonality (Сезонність)

Аналіз найкращих годин та днів для торгівлі:

1. Перейдіть в **Seasonality**
2. **Hourly Heatmap** — 24 колонки (кожна година доби):
   - Зелена = в середньому ціна зростає в цю годину
   - Червона = в середньому падає
3. **Day of Week** — які дні тижня найприбутковіші

---

## Market Breadth (Ширина ринку)

Здоров'я всього крипторинку:

1. Перейдіть в **Market Breadth**
2. **Breadth Score** (0-100):
   - > 70 = Strong Bull (більшість монет зростає)
   - 30-70 = Neutral
   - < 30 = Weak (більшість падає)
3. **Advancing / Declining** — скільки монет зростає vs падає
4. **% Above SMA20** — скільки монет вище своєї середньої
