# Налаштування та акаунт

## Settings (Налаштування)

Перейдіть в **Settings** через бічну панель або натисніть на аватар.

### Профіль

- **Display Name**: ваше ім'я на платформі
- **Email**: email (тільки для читання)
- **Timezone**: ваш часовий пояс
- Натисніть **"Save Changes"** після редагування

### Зовнішній вигляд

- **Dark** — темна тема (чорний фон, золоті акценти)
- **Light** — світла тема (кремовий фон)
- Переключити також можна іконкою ☀️/🌙 в хедері

### Мова

- **English** — доступна
- **Ukrainian** — доступна
- **German** — доступна
- **Spanish** — доступна

## Двофакторна автентифікація (2FA)

### Навіщо потрібна

2FA додає другий рівень захисту. Навіть якщо хтось дізнається ваш пароль — без коду з телефону не зможе увійти.

### Як увімкнути

1. В Settings знайдіть секцію **"Two-Factor Authentication"**
2. Натисніть **"Enable 2FA"**
3. З'явиться:
   - **Secret Key** — 32-символьний код
   - **QR Code URL** — для сканування
4. Відкрийте Google Authenticator (або Authy) на телефоні
5. Додайте акаунт → введіть Secret Key вручну або відскануйте QR
6. Введіть 6-значний код з додатку в поле "Verification Code"
7. Натисніть **"Verify"**
8. Статус змінюється на "Enabled" ✅

### Важливо

- Збережіть Secret Key в безпечному місці
- Без нього ви не зможете відновити доступ

## Telegram

Отримуйте сповіщення про алерти та сигнали прямо в Telegram.

### Як підключити

1. В Settings знайдіть секцію **"Telegram"**
2. Відкрийте Telegram → знайдіть бота Quantis (@QuantisBot)
3. Надішліть боту команду `/start`
4. Бот надасть вам **Chat ID**
5. Введіть Chat ID в поле на сайті
6. Натисніть **"Connect"**
7. Натисніть **"Send Test"** для перевірки

### Що приходить в Telegram

- Спрацювання алертів (ціна досягла рівня)
- Нові торгові сигнали BUY/SELL
- Тижневий звіт (Weekly Report)

## Безпека

Quantis захищає ваші дані:
- **Паролі**: хешовані bcrypt (12 rounds)
- **Токени**: JWT з 15-хвилинним терміном
- **API Keys**: AES-256-GCM шифрування
- **2FA**: TOTP (Google Authenticator)
- **Rate Limiting**: захист від DDoS
- **HTTPS**: TLS 1.3 шифрування
- **No tracking**: ми не продаємо ваші дані
