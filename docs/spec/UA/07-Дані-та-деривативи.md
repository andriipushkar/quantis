# Дані, деривативи та аналітика ринку

## Funding Rates (Ставки фінансування)

Ставки фінансування ф'ючерсних контрактів — індикатор настрою ринку.

1. Перейдіть в **Funding Rates**
2. Таблиця показує для кожної пари:
   - **Rate**: поточна ставка
     - **Зелена (negative)** — шорти платять лонгам (ведмежий настрій)
     - **Червона (positive)** — лонги платять шортам (бичачий настрій)
   - **Annualized** — річна ставка
   - **Next Funding** — зворотний відлік до наступного розрахунку
   - **Prediction** — прогноз напрямку
3. Extreme rates (> 0.05%) виділені — можливість для арбітражу

## Open Interest (Відкритий інтерес)

Сума всіх відкритих ф'ючерсних позицій.

1. Перейдіть в **Open Interest**
2. Summary: загальний OI, 24H зміна
3. Таблиця: Symbol, OI ($M), 24H%, Volume, OI/Vol Ratio
4. **Divergence flags**: коли ціна зростає а OI падає (або навпаки) — це попередження

**Як читати:**
- Ціна зростає + OI зростає = сильний тренд
- Ціна зростає + OI падає = слабкий ралі (закриття шортів)
- Ціна падає + OI зростає = ведмежий тиск

## Intermarket Analysis (Міжринкова аналітика)

Як традиційні ринки впливають на крипто.

1. Перейдіть в **Intermarket Analysis**
2. Активи по групах:
   - **Індекси**: S&P 500, NASDAQ
   - **Товари**: Золото, Срібло, Нафта
   - **Облігації**: US 10Y, 2Y Yield
   - **Валюти**: DXY (індекс долара)
3. **BTC Correlation**: як BTC корелює з кожним активом
   - DXY від'ємна кореляція = долар зростає → крипто падає
   - Gold позитивна = обидва "safe haven"
4. **Risk-On / Risk-Off** badge

## Tokenomics (Токеноміка)

Аналіз пропозиції та інфляції токенів.

1. Перейдіть в **Tokenomics**
2. Виберіть монету (BTC, ETH, SOL, BNB, XRP)
3. Для кожного:
   - **Supply Gauge**: Circulating / Max Supply
   - **Inflation Rate**: річна інфляція
   - **FDV**: Fully Diluted Valuation
   - **Score** (0-100): загальна оцінка токеноміки
   - **Unlock Schedule**: коли розблокуються нові токени

**Оцінки:**
- BTC: 95/100 (93% в обігу, 1.7% інфляція)
- ETH: 88/100 (дефляційний після Merge)
- SOL: 65/100 (високий стейкінг, 5.5% інфляція)

## On-Chain Analytics (On-chain аналітика)

Об'єднана сторінка з 2 вкладками: Dev Activity та Network Metrics.

1. Перейдіть в **On-Chain Analytics** (`/on-chain`)

### Network Metrics (Мережеві метрики)

On-chain здоров'я блокчейнів.

1. На вкладці **Network Metrics** виберіть: BTC / ETH / SOL
2. Метрики:
   - **DAA**: Daily Active Addresses (активні гаманці/день)
   - **Tx Count**: кількість транзакцій
   - **Transfer Value**: обсяг переказів в USD
   - **NVT Ratio**: Network Value to Transactions (як P/E для крипто)
   - **Health Score**: загальне здоров'я мережі

## Bitcoin Models (Моделі оцінки BTC)

5 моделей для оцінки "справедливої" ціни BTC.

1. Перейдіть в **BTC Models**
2. Поточна ціна BTC + Overall Consensus
3. Моделі:
   - **Stock-to-Flow**: ~$120,000 (based on scarcity)
   - **Rainbow Chart**: в якій "зоні" зараз BTC
   - **Pi Cycle Top**: чи є сигнал топу циклу
   - **MVRV Z-Score**: оцінка відносно реалізованої ціни
   - **Power Law**: логарифмічна регресія

Кожна показує: fair value, deviation від поточної ціни, сигнал (undervalued/fair/overvalued)

> **Увага:** моделі НЕ гарантують майбутніх цін. Використовуйте як один з інструментів аналізу.

## Token Scanner (Сканер ризиків)

Оцінка безпеки токена перед купівлею.

1. Перейдіть в **Token Scanner**
2. Введіть символ (наприклад BTCUSDT)
3. Натисніть **Scan**
4. Результат: Score (0-100) + Label:
   - **SAFE** (80-100, зелений)
   - **CAUTION** (60-79, жовтий)
   - **RISKY** (40-59, оранжевий)
   - **DANGER** (0-39, червоний)
5. 6 факторів: Liquidity, Data History, Volatility, Volume Consistency, Price Stability, Exchange Presence
