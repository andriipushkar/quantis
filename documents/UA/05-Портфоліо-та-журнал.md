# Портфоліо та торговий журнал

## Portfolio (Портфоліо)

Відстежуйте свої криптоактиви в одному місці.

### Огляд

1. Перейдіть в **Portfolio**
2. Зверху: Total Value, 24H Change
3. **Donut Chart** — візуальний розподіл по монетах
4. **Positions Table** — деталі по кожному активу:
   - Asset, Holdings, Current Price, Market Value, 24H%, Allocation %

### Demo режим

Поки ви не підключили біржовий API, працює демо-портфоліо:
- BTC 0.5, ETH 3.0, SOL 10, BNB 2
- Ціни оновлюються в реальному часі

### Export CSV

Натисніть **"Export CSV"** щоб завантажити дані в Excel/Google Sheets.

### Rebalancing (Ребалансування)

1. Прокрутіть до секції **"Portfolio Rebalance"**
2. Встановіть цільові відсотки: наприклад BTC 50%, ETH 30%, SOL 15%, BNB 5%
3. Система покаже:
   - **Current vs Target** — бари порівняння
   - **Suggestions** — "Продайте $X BTC, купіть $X ETH"
4. Перевірте що сума = 100%

## Trading Journal (Торговий журнал)

Записуйте та аналізуйте кожен свій трейд.

### Навіщо потрібен журнал?

- Відслідковувати свій прогрес
- Розуміти які стратегії працюють
- Контролювати емоції
- Покращувати Win Rate

### Як додати трейд

1. Перейдіть в **Journal**
2. Натисніть **"Add Trade"**
3. Заповніть:
   - **Pair**: монета (наприклад BTCUSDT)
   - **Direction**: Long (купівля) або Short (продаж)
   - **Entry Price**: ціна входу
   - **Exit Price**: ціна виходу (якщо вже закрито)
   - **Size**: розмір позиції в USD
   - **Strategy**: яку стратегію використали
   - **Emotional State**: ваш емоційний стан
     - Calm (спокійний) — ідеально
     - FOMO (страх пропустити) — небезпечно
     - Revenge (помста) — дуже небезпечно
     - Greedy (жадібність) — обережно
     - Fearful (страх) — може бути корисним
   - **Confidence**: 1-5 зірок (наскільки впевнені в трейді)
   - **Notes**: нотатки (що бачили, чому увійшли)
4. Натисніть **"Save"**

### Статистика

Stats Bar зверху показує:
- **Total Trades** — загальна кількість
- **Win Rate** — % прибуткових
- **Avg Win** — середній прибуток
- **Avg Loss** — середній збиток
- **Profit Factor** — сума прибутків / сума збитків (> 1.0 = прибуткова торгівля)

### Аналіз журналу

Зверніть увагу на:
- Чи є зв'язок між емоціями та результатом? (FOMO = збитки?)
- Яка стратегія найприбутковіша?
- Яка впевненість (1-5) найточніша?

## Tax Report (Податковий звіт)

Розрахунок податків по крипто-операціях.

### Як використати

1. Перейдіть в **Tax Report**
2. Виберіть рік (2026)
3. Бачите:
   - **Total Gains** (зелений) — сума прибутків
   - **Total Losses** (червоний) — сума збитків
   - **Net P&L** — чистий результат
   - **Total Trades** — кількість операцій

### Деталі

- **By Asset** — P&L по кожній монеті окремо
- **Trade History** — повна таблиця операцій
- **Tax Method**: FIFO (First In, First Out) — активний

### Експорт

Натисніть **"Download CSV"** для вашого бухгалтера.

> **Важливо:** Це оцінка. Для точних податкових розрахунків консультуйтесь з фінансовим радником.

## Wallet Tracker (Відстеження гаманців)

Моніторинг крипто-гаманців на блокчейні.

### Як додати гаманець

1. Перейдіть в **Wallet Tracker**
2. В формі "Add Wallet":
   - **Address**: адреса гаманця (0x... для Ethereum)
   - **Chain**: Ethereum / Solana / Bitcoin
   - **Label**: назва для зручності (наприклад "My Hot Wallet")
3. Натисніть **"Track"**

### Перегляд

- Бачите: адресу (скорочену), мережу, мітку, загальну вартість
- Натисніть на гаманець → розгортається таблиця holdings (токени, суми, вартість)

### Видалення

Натисніть **"Remove"** біля гаманця щоб прибрати зі списку.
